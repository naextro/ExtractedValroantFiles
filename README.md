# Valorant Audio Showcase

A clean, dark-themed web gallery for displaying and playing extracted audio files from Valorant. Perfect for showcasing Valorant sound effects, music tracks, voice lines, and game audio in an organized, searchable interface.

## 🎵 Features

- **Dark Theme Design**: Modern, Valorant-inspired dark UI with gradients and shadows
- **Audio Playback**: Built-in HTML5 audio player for each track
- **Search Functionality**: Real-time search by filename or track title
- **Sorting Options**: Sort tracks alphabetically (A-Z, Z-A) or by date
- **Responsive Layout**: Mobile-friendly design that works on all devices
- **Automatic Metadata**: Displays track duration, file size, and modification dates
- **GitHub Pages Ready**: Static HTML that deploys easily to GitHub Pages
- **No Dependencies**: Pure HTML, CSS, and JavaScript - no frameworks required

## 🎮 Usage

1. **Browse Tracks**: Scroll through the audio gallery
2. **Search**: Use the search bar to find specific tracks by name
3. **Sort**: Choose sorting options from the dropdown
4. **Play Audio**: Click play on any audio player to listen
5. **Reset**: Clear search and return to default sorting
